# Program 10a - HTTP Request and Response Handling

## Objective
Develop a servlet program to demonstrate handling of HTTP Request and HTTP Response using user input form.

## Files
- UserDataServlet.java
- index.jsp
- result.jsp

## Source Code
- Servlet: https://github.com/musabnova/Program10a/blob/master/src/main/java/com/HTTP_REQ_RES/UserDataServlet.java
- Input Page: https://github.com/musabnova/Program10a/blob/master/src/main/webapp/index.jsp
- Output Page: https://github.com/musabnova/Program10a/blob/master/src/main/webapp/result.jsp

## Concept Used
- HttpServletRequest
- HttpServletResponse
- Form handling
- Request parameters
- JSP for displaying output

## How it Works
1. User enters data in index.jsp
2. Form submits data to servlet
3. Servlet processes request
4. Data is forwarded to result.jsp
5. Output is displayed to user

## How to Run
1. Deploy project on Apache Tomcat
2. Open:
   http://localhost:8080/Program10a/
3. Enter user details and submit form
4. View processed output
